jQuery Dimension Methods

jQuery has several important methods for working with dimensions:

  • width() - sets/returns the width of an element (excludes padding, border, margin)
  • height() - sets/returns the height of an element (excludes padding, border, margin)
  • innerWidth() - sets/returns the width of an element (includes padding)
  • innerHeight() - sets/returns the height of an element (includes padding)
  • outerWidth() - sets/returns the width of an element (includes padding & border)
  • outterHeight() - sets/returns the height of an element (includes padding & border)
  • outerWidth(true) - returns the width of an element (includes padding, border & margin)
  • outterHeight(true) - returns the height of an element (includes padding, border & margin)
