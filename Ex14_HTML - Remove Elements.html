jQuery - Add Elements

With jQuery, it is easy to remove existing elements/content

Remove Elements / Content

To remove elements and content, there are mainly 2 jQuery methods

  • remove() - Removes the selected element (and its child elements)
  • empty() - Removes the child elements from the selected elements

Filter the Elements to be Removed

The jQuery remove() also accepts one parameter, which allows you to filter the elements to be remvoed.

The parameter can be any of hte jQuery selector syntaxes

The following example removes all <p> elements with class="test"

$( "p" ).remove( ".test" );

The following example removes all <p> elements with class="test" and class="demo"

$( "p" ).remove( ".test" , ".demo" );
