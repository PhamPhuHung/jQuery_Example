More Examples of jQuery Selectors

Syntax Description
$("*") Selects all elements
$(this) Selects the current HTML element
$("p.intro") Selects all <p> elements with class="intro"
$("p:first") Selects all the first <p> elements
$("ul li:first") Selects the first <li> element of the first <li>
$("ul li:first") Selects the first <li> element of the first <ul>
$("ul li:first-child") Selects the first <li> element of every <ul>
$("[href]") Selects all elements with an href attribute
$("a[href='_blank']") Selects all <a> elements with a target attribute value equal to "_blank"
$("a[href!='_blank']") Selects all <a> elements with a target attribute value NOT equal to "_blank"
$(":button") Selects all <button> elements and <input> elements of type="button"
$("tr:even") Selects even <tr> elements
$("tr:odd") Selects odd <tr> elements
