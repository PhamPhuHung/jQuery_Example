jQuery Fading Methods

With jQuery, you can fade an element in and out of visibility

jQuery has th following fde methods:

  • fadeIn()
  • fadeOut()
  • fadeToggle()
  • fadeTo()

Syntax

$(selector).fadeIn(speed, callback);
$(selector).fadeOut(speed, callback);
$(selector).fadeToggle(speed, callback);

The optional speed parameter specifies the duration of the effect. It can take the following values: "slow", "fast, or millisecond

The optional callback parameter is a function to be executed AFTER the fading completes


$(selector).fadeTo(speed, opacity, callback);

The required speed parameter specifies the duration of the effect. It can take the following values: "slow", "fast", or milliseconds

The required opacity parameter specifies fading to a given opacity (values between o & 1)

The optional callback parameter is a function to be executed after the function completed

jQuery fadeIn() & fadeOut Method

The jQuery fadeIn method is used to fade in a hidden element

The jQuery fadeOut method is used to fade out a visible element


jQuery fadeToggle

The jQuery fadeToggle method toggles between the fadeIn() & fadeOut methods

If the elements are faded out, the fadeToggle will fade them in

If the elements are faded in, the fadeToggle will fade them out


jQuery fadeTo()

The fadeTo() method allows fading to a given opacity (value between 0 & 1)
