jQuery Animations - The animate() Methods

The jQuery animate() method Ã­ used to create custom animations

Syntax

$(selector).animate({params}, speed, callback);

The required params parameter defines the CSS properties to be animated

The optional speed parameter specifies the duration of the effect. It can take the following values: "slow", "fast, or millisecond

The optional callback parameter is a function to be executed AFTER the fading completes

The following example demonstrates a simple use of the animate() method; it moves <div> element to thr right, until it has reached a left property of 250px:

$( "div" ).click( function (){
  $( "div" ).animate({left: '250px' });
});

By default, all HTML elements have a static position, and cannot be moved.
To manipulate the position, remember to first set the CSS position property of the element to relative, fixed, or absolute


jQuery animate - Manipulate Multiple Properties

Notice that multiple properties can be animated at the same time:

$( "div" ).click( function (){
  $( "div" ).animate({
     left: '250px' ,
     opacity: '0.5' ,
     height: '150px' ,
     width: '150px'
  }, 1000);
});

Is it possible to manipulate ALL CSS properties ith the animate() method?
Yes, almose! However, there is 1 important thing to remember: all property names must be camel-cased when used with the animate() method: YOu will need to write paddingLeft instead of padding-left, margin-Right instead of margin-right, and so on.

Also, color animation is not included in the core jQuery library. If you want to animate color, you need to download the Color Animations plugins from jQuery.com


jQuery animate() - Using Relative Values

It is also possible to define relative values (the value is then relative to the element's current value). This is done by putting += or -= infront of the value:

$( "div" ).click( function (){
  $( "div" ).animate({
     left: '+=50px'   }, 1000);
});


jQuery animate() - Using Pre-defined Values

You can even specify a property's animation value as "show", "hide", "toggle"

$( "div" ).click( function (){
  $( "div" ).animate({
     left: 'toggle'   }, 1000);
});
